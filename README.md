# 🚀 撸毛项目追踪器

一个强大的空投项目管理和X内容生成工具，帮助Crypto撸毛党高效追踪项目、管理投资，并一键生成X平台推广内容。

## ✨ 核心功能

### 1. 项目管理
- ✅ 完整的CRUD操作（创建、读取、更新、删除）
- 📊 投资与收益追踪
- 📈 ROI自动计算
- 📅 任务进度追踪
- ⏰ 截止日期与提醒功能
- 🏷️ 多维度标签（链条、类别、潜力、状态）

### 2. 数据统计
- 📊 实时仪表盘
- 💰 总投入/收益统计
- 📈 ROI可视化
- 📋 项目分类汇总

### 3. X内容生成器
一键生成以下类型内容：
- 📅 **今日机会**：汇总当前撸毛项目
- 📊 **ROI报告**：展示收益和投资回报
- 🧵 **完整Thread**：生成项目组合Thread
- 🔍 **项目亮点**：单个项目详细分析

### 4. 活动日志
- 📜 自动记录所有操作
- 🔍 提醒系统
- 📊 操作历史追踪

## 🛠️ 技术栈

- **后端**: Flask + SQLAlchemy
- **数据库**: SQLite
- **前端**: HTML + CSS + JavaScript
- **定时任务**: APScheduler

## 📦 安装与运行

### 1. 安装依赖
```bash
cd airdrop-tracker
pip install -r requirements.txt
```

### 2. 运行应用
```bash
python app.py
```

### 3. 访问应用
打开浏览器访问: `http://localhost:5000`

## 📖 使用指南

### 添加项目
1. 点击右上角"+ 新增项目"按钮
2. 填写项目信息：
   - 基本信息（名称、描述）
   - 链条和类别
   - 投资和预期收益
   - 任务进度
   - 截止日期和提醒
3. 点击"保存"

### 生成X内容
1. 在右侧"X内容生成器"面板
2. 选择内容类型：
   - **今日机会**：生成所有进行中项目的汇总
   - **ROI报告**：生成收益分析报告
   - **完整Thread**：生成项目组合Thread
3. 点击"📋 复制"按钮
4. 粘贴到X平台发布

### 查看统计
- 顶部仪表盘显示关键指标
- 实时更新项目数据
- ROI自动计算

## 🎯 使用场景

### 场景1：每日推广
每天早上生成"今日机会"推文，展示你正在追踪的项目，吸引关注。

### 场景2：收益分享
申领到空投后，生成"ROI报告"推文，展示你的投资回报率。

### 场景3：项目分析
为单个项目生成详细分析推文，分享你的见解。

### 场景4：月度总结
生成完整Thread，回顾本月的撸毛组合和表现。

## 🔧 自定义配置

### 修改端口
在 `app.py` 最后修改：
```python
app.run(debug=True, host='0.0.0.0', port=8000)
```

### 修改数据库
在 `app.py` 中修改数据库URI：
```python
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///custom_name.db'
```

## 📊 数据备份

数据库文件位于: `airdrop_tracker.db`

备份方法：
```bash
# 备份数据库
cp airdrop_tracker.db airdrop_tracker_backup_$(date +%Y%m%d).db
```
